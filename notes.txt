node_modules directory is where the project's dependencies will be located including the React library itself. Any other packages that need to be installed will also be located in this directory.

public directory is where the project's files that are visible to the browser will be located including index.html.

src directory is where 99% of the project's code will be located

    index.jsx is what kickstarts our project. It takes all the React components and renders them to the #root div in index.html through DOM(See index.jsx ln 6).

    Script.jsx is an example of React component

package.json lists all the dependencies of the project, it also lists the scripts of the project
